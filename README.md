# 3D Bouncing Ball Animation

## Overview

This project is a simple 3D bouncing ball animation built to demonstrate basic physics simulation and real-time rendering concepts. The animation models gravity, velocity, and ground collision to produce realistic bouncing motion. The project uses mathematical formulations to model 3d objects and movements

## Features

* 3D ball rendering
* Gravity-based motion
* Ground collision detection
* Realistic bounce behavior

## Technologies Used

* C++
* OpenGL / WebGL-style rendering pipeline (depending on implementation)
